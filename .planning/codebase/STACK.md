# Tech Stack - MiniFootballLeagueAnalyzer

## Backend (Data Pipeline)
- **Language**: Python 3.10+
- **Scraping**: Selenium, BeautifulSoup4, Requests
- **Data Processing**: NumPy, Python JSON module
- **Testing**: Pytest

## Frontend (Web Application)
- **Framework**: Astro 6 (Islands Architecture)
- **Library**: React 19
- **Styling**: Vanilla CSS, Framer Motion (Animations), Lucide React (Icons)
- **Visualization**: Chart.js, react-chartjs-2
- **Maps**: Mapbox GL, react-map-gl
- **State Management**: Nanostores

## Services & Infrastructure
- **Database/Auth**: Supabase (PostgreSQL, Auth)
- **AI**: Google Generative AI (Gemini SDK)
- **Hosting**: Vercel
- **CI/CD**: GitHub Actions (Weekly Scraper)
